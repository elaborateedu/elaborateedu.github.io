What is Elaborate?
A hub for every
student and teacher.
Elaborate is a free educational platform where students pin their favourite apps, join classrooms, and build a personalised learning hub. Teachers use it to push curated app lists, send announcements, and track engagement.
📚
App Library
A curated directory of 40+ vetted educational tools. Students can browse, filter by subject, and add apps to their hub in one tap. Getting listed means permanent discoverability.

Classroom Integration
Teachers can push app collections to all students in their classroom at once. If your app is in the library, teachers can instantly surface it to their entire class.

How the App Library Works
Each app in the Elaborate library is a simple record stored in Firestore with a name, URL, description, category, and icon. When a student taps your app, Elaborate opens your URL. There is no iframe embedding — your app runs natively in the browser.
Teachers can push your app to all students in their classroom from their dashboard. Students can also browse the Discover tab and add your app themselves with a single tap.
Supported launch modes
New tab (default) Same tab (opt-in)
Example App Record
{ "name": "Desmos", "url": "https://desmos.com/calculator", "desc": "Powerful graphing calculator", "category": "Math", "ageRange": "6–12", "approved": true, "featured": false, "privacyUrl": "https://desmos.com/privacy", "submittedBy": "you@yourcompany.com" }
Classroom Push API (Beta)
Teachers on Elaborate can push a collection of apps to all students in their classroom. When your app is part of a pushed collection, it appears at the top of each student's hub pinned automatically.
There is no API key required for this — it is entirely teacher-initiated from within the Elaborate classroom dashboard. Your app just needs to be listed in the library.

If you want to build a deeper integration — like passing a student identifier or syncing progress — get in touch. We are exploring authenticated handoffs for verified partners.

Data Schema
Firestore collections.
Elaborate uses Firebase Firestore. Here is a high-level reference of the public-adjacent collections that relate to app library integration and classroom use.
appLibrary / {appId}
The approved app library. Read by all students; write by Elaborate admins only.
FieldTypeDescription
namestringDisplay name of the app
urlstringFull HTTPS URL the app opens to
descstringOne-line description (max 120 chars)
categorystringe.g. "Math", "Science", "Coding"
ageRangestringe.g. "K–12", "6–8"
approvedbooleanOnly true apps are shown in the Discover tab
featuredbooleanFeatured apps appear at the top of the library
privacyUrlstringLink to app's privacy policy
submittedAttimestampWhen the submission was received
users / {uid}
Student and teacher profiles. Private — only readable by the owner.
FieldTypeDescription
firstName / lastNamestringDisplay name, set during onboarding
rolestring"student" | "teacher" | "district_admin"
appsarrayPinned apps array — each has name, url, color, category
classroomIdstring?ID of the classroom the student is enrolled in
streaknumberConsecutive daily login streak
favoriteIdsarrayApp IDs the user has starred
classrooms / {classroomId}
Teacher-owned classroom documents. Readable by enrolled students.
FieldTypeDescription
namestringClassroom display name
teacherId / teacherNamestringUID and display name of the teacher
joinCodestring6-character code students use to join
studentsarrayArray of enrolled student UIDs
appRequestssubcollectionApps the teacher has pushed to the class
